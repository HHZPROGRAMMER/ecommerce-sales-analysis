# 📊 Ecommerce Sales & Profit Analytics

An interactive and dynamic end-to-end Data Analytics project that covers Data Cleaning, Exploratory Data Analysis (EDA), and interactive Dashboard design using Python and Power BI.

## 🎯 Project Workflow
1. **Data Cleaning & EDA:** Raw data was cleaned, missing values handled, and initial statistical insights derived using **Python (Pandas)**.
2. **Exploratory Visualizations:** Trends and distributions were plotted using **Matplotlib & Seaborn** (Available in the `notebooks/` folder).
3. **Interactive Dashboard:** Designed a professional, dark-themed dashboard in **Power BI** to enable deep-dive business tracking.

## 💡 Key Business Insights
* **Monthly Sales Trends:** Sales show stable performance throughout the year with specific seasonal peaks.
* **Regional Distribution:** North and East regions dominate the sales volume compared to the West and South.
* **Payment Preferences:** Payment methods (Net Banking, COD, Credit/Debit Cards, UPI) share almost equal distribution, ensuring flexible transaction options for customers.
* **Profitability:** Specific sub-categories like Sofas and Lamps show healthy profit benchmarks.

## 🛠 Tech Stack
* **Language:** Python 3
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
* **BI Tool:** Power BI Desktop (DAX, Power Query)
